Nell’esempio qui sopra potete vedere che con l’attributo htmlTag="div" trasformo l’elemento in un div, con l’attributo htmlClass="my-custom-css-class" gli assegno le classi di cui ho bisogno e, opzionale, posso assegnare un id con questo attributo htmlId="my-custom-css-id".
