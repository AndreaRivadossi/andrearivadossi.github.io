Trasferire file tra server

2021/05/03 Procedure

Sicuramente vi sarà capitato di dover duplicare ambienti di sviluppo per collegi, collaboratori o per voi stessi oppure di dover trasferire i contenuti della media o il dump del DB aggiornato dalla produzione del sito a cui state lavorando.

Indipendentemente dalla piattaforma, io utilizzo questi 3 step:

  • Comprimo i file o la cartella che li contiene,
  • Trasferisco la cartella zippata,
  • Decomprimo.

Comprimere

Per comprimere dei file o una cartella utilizzo il comando zip, in questo modo:

zip -r nome-file-zippato.zip nome-cartella-da-zippare/

Trasferire

Principalmente utilizzo due metodi per trasferire i file.

WGET

Utilizzare questo metodo se si ha il link diretto al file necessario, in questo modo:

wget http://example.com/nome-file.zip

In caso la directory fosse protetta da password utilizzare questa sintassi:

wget --user nome-utente --password la-mia-password http://example.com/nome-file.zip

SCP

Utilizzare questo metodo se si hanno gli accessi SSH al server di destinazione.

scp nome-file.zip nome-utente@ip-macchina:/percorso/di/destinazione/nel/server

Dopo aver lanciato questo comando verrà richiesta la password, inseritela e inizierà il trasferimento.

Decomprimere

Per decomprimere la cartella basterà utilizzare il comando unzip:

unzip nome-file-zippato.zip
